## Overview

Photography portfolio and business website for Simon Wickes, progressing from foundational infrastructure through the core portfolio showcase, conversion paths, blog, and finally SEO optimization. The image pipeline was established first because every subsequent phase depended on optimized image delivery.

## Milestone Summary

**Key Decisions:**

- Astro over WordPress: Performance, security, developer control
- Tailwind v4 CSS-first tokens: No config file, @theme directives
- Markdown over Strapi CMS: Shared hosting can't run Node.js
- Third-party gallery service (Pic-Time): Purpose-built for photographers
- Static site on shared hosting: Uses existing InMotion hosting
- astro-swiper custom element: Self-initializing, no client:load needed
- PhotoSwipe for lightbox: Keyboard + touch + View Transitions compatible
- Resend for email: Simple API, works with Astro Actions
- Fuse.js for search: Client-side fuzzy search, no server needed
